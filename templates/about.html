How the system works

1. Input material

User uploads a PDF/DOCX file or pastes educational text. The system extracts and normalizes textual content.

2. Text processing

The text is preprocessed, cleaned, and limited to ensure stable generation.

3. Question generation

AI models generate open-ended questions or multiple-choice tests based on extracted concepts.

4. Export & storage

Generated exams can be exported to PDF or DOCX format and stored in history.
